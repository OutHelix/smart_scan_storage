# Smart Scan Storage

Версия 0.0.1. Хранилище сканов: OCR, поиск, склейка страниц.

Backend: FastAPI. Frontend: React, TypeScript, Vite.

## Запуск

**Backend**

1. `cd backend`
2. `python -m venv .venv`
3. `.venv\Scripts\activate`
4. `pip install -r requirements.txt`
5. `python run.py`

http://localhost:8000

**Frontend**

1. `cd frontend`
2. `npm install`
3. `npm run dev`

http://localhost:5173
